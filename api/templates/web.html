FDriver 是一款强大的桥接工具。通过 Python 驱动层与配套的 Scratch 扩展积木，让你的编程作品轻松读写本地文件、调用外部 API，甚至控制硬件。

下载 FDriver 引擎 查看积木扩展

三位一体的工作流

  • 1

    Python 后端 (Server)

    在本地静默运行，负责处理高权限系统指令。

  • 2

    FDriver 积木扩展

    通过浏览器请求与后端通信，操作简单直观。

  • 3

    本地文件/网络 API

    突破浏览器沙箱，实现真正的持久化存储和数据交换。
